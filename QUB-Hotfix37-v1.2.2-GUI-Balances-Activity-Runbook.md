# QUB Hotfix37 / v1.2.2 — GUI balances, activity filters, send speed

This release is GUI-only.

- No chain upgrade
- No activation height changes
- No consensus changes
- Based on v1.2.0 mainnet/testnet configuration, not v1.2.1 testnet-only naming

Key changes:

- Send/QNS registration no longer run a blocking full P2P sync before transaction creation.
- Start/stop mining icons are slightly larger.
- Live chain data removes wallet-balance rows and adds supply rows plus info tooltips.
- Wallet/address section renamed to Create / import wallet.
- Import private key GUI added.
- Address balances panel added next to Address Activity.
- QUB/JIN crypto cards added with buttons for future marketplace/infusion/melt flows.
- QNS tab added with register and per-name action buttons.
- Address Activity adds Direction and Asset filters; Type includes Infusion/Melt/Conversion.
