# Contributing

QUB Core is still in an early stabilization phase.

Useful contributions include:

- bug reports
- reproducible sync/mining issues
- pool mining reports
- stale/orphan behavior reports
- performance observations
- UI/UX fixes
- documentation
- code review
- security review

## Bug report minimum

Please include:

- QUB Core version
- OS
- CPU/GPU
- mining mode: solo or pool
- screenshot
- exact status text
- recent blocks top rows
- peers/block stream rows
- steps to reproduce

## Pull requests

Before large pull requests, open an issue or discussion first.

Do not submit code that includes:

- private keys
- wallet files
- user data
- build artifacts
- installers
- binaries
- local chain data
